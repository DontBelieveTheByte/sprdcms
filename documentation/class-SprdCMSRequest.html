Class SprdCMSRequest

Helper class for the front controller is used to decompose request parameters and transform it into a command with parameters.

Helper class for the front controller is used to decompose request parameters and transform it into a command with parameters. The $_GET superglobal is evaluated and exploded into an array. If nothing is passed by $_GET a default command is usually written.

# extract( )

This method does several things to in order to extract a valid command from the $_GET superglobal. It starts by checking if clean URLs are enabled and if it this is the case, it will use $_SERVER['REDIRECT_QUERY_STRING'] as its input. Otherwise it uses the $_GET superglobal and uses a regular expression to explode the query string into an array. The is a special case for handling the testing of clean URLs capability. If nothing can be extracted from the previous cases, it will explicitely return NULL.
