Class SprdCMSCommandMapper

Transaction script to control most of the flow of execution.

A transaction script that resolves requests to commands and also delegates to another SprdCMSProcessor interface object for ajax requests or others.

SprdCMSCommandMapper implements SprdCMSProcessor

public
# process( array $params = NULL )

An numericall indexed array is needed as input. The first index is the command to execute and subsequent indexes are parameters. Commands are whitelisted using a switch block that validates known commands, so any other unkown command is rejected. Implements the SprdCMSProcessor interface, can also delegate requests to other SprdCMSProcessor objects.
