Class SprdCMSDb

A PDO wrapper class used for handling database operations that is implemented as a singleton.

A PDO wrapper class used for handling database operations that is implemented as a singleton. The class uses an application wide configuration to build a data source name (DSN) and initialize a PDO instance. The constructor is private and can only be called via the static method getInstance. getInstance returns the same database handle throughout the application.
