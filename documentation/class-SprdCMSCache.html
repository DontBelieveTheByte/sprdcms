Class SprdCMSCache

This class is in charge of handling the caching mechanism.

This class is in charge of handling the caching mechanism. It decorates the SprdCMSCommandMapper class by first checking if there's a cache hit or a cache expiration. Implementation of the SprdCMSProcessor interface garanties seamless delegation to SprdCMSCommandMapper in order to process the request when the cache doesn't exist or is expired.

SprdCMSCache implements SprdCMSProcessor
