Class SprdCMSSiteConfig

Provides application wide configuration values.

A helper class that is used to provide application wide configuration values as a registry of run time settings. Most configuration values are loaded from an XML file located at includes/config.xml. Implemented as a singleton to use a single configuration. There is no locking mechanism since only a single user can modify the configuration.
