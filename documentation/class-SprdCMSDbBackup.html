Class SprdCMSDbBackup

Provides the functionality of backing up the MySQL database.

Provides the functionality of backing up the MySQL database. It will only work where the MySQL database runs on the same host as the webserver and PHP has shell access. Implements SprdCMSProcessor interface when the request is determined to be delegated to this object.

SprdCMSDbBackup implements SprdCMSProcessor

The delegation methods builds the backup command using the configuration file. It saves a database dump to disk and sends it back to the user's browser a a downloadable file. This is achieved by handling the request directly and settings the http protocol header.
