Classes summary
SprdCMSAjaxController This class is in charge of handling all the ajax request.
SprdCMSBackStore Class is responsible for administering the SprdCMS backend.
SprdCMSCache This class is in charge of handling the caching mechanism.
SprdCMSCommandMapper Transaction script to control most of the flow of execution.
SprdCMSDb A PDO wrapper class used for handling database operations that is implemented as a singleton.
SprdCMSDbBackup Provides the functionality of backing up the MySQL database.
SprdCMSFrontController All request to the application must go through the front controller.
SprdCMSInstall The install script.
SprdCMSItem This is base class that defines what properties a store items should hold and its common methods.
SprdCMSItemCSV Class useful for instantiating store articles built from a CSV file.
SprdCMSItemManual This class create instances of a valid store articles build from manual input of a unique spreadshirt article ID.
SprdCMSLogger Central logging facility.
SprdCMSRequest Helper class for the front controller is used to decompose request parameters and transform it into a command with parameters.
SprdCMSRewrite This class is in charge of enabling or disabling URL rewriting.
SprdCMSSession Simple class used to control session of the site administator.
SprdCMSSiteConfig Provides application wide configuration values.
SprdCMSStore Abstract base class to define common admin and client store functionality.
SprdCMSStoreFront Provides client side store functionality by extending SprdCMSStore base class.
SprdCMSViewControl This class handles finding and loading and redenring of different views.
